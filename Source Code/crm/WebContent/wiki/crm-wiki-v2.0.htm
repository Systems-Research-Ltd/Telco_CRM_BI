How to Login User

Overview

The User Login window is provided to allow the user to login to the application using his/her password. The user can also select the application here.

How to Login?

1.      Enter User Name provided by the account manager.

2.      Enter Password provided by the account manager.

3.      Choose the application you want to login BI or CRM.

4.      Press Login. You will see the dashboard.

Description of all input fields on this page as under:

Login Name/Email

Enter the valid Login name/email.

Password

Enter the right password.

Login

Press this button to login.

Forgot Password

Press this button to recover forgotten password.

Marketing

Overview

The Marketing window is provided to show a list of all the marketing campaigns launched by the Company. However, the user should first select the company name from the panel on the left hand side of the screen.

Description of all input fields on this page as under:

Details Tab

A form will appear showing user details.

Settings Tab

Campaign launch settings can be done from here.

Leads

Leads affiliated to each campaign are shown.

Edit

Data can be edited by clicking on edit button.

Delete

Campaign can be deleted by pressing delete button. Only after pressing the delete confirmation button campaign can be deleted.


 

Create Campaign

Overview

The Create Campaign window is provided to create a new marketing campaign for the respective company.

How to create a campaign?

1.      Click on marketing Tab.

2.      Press  icon to open Create New Marketing Campaign Form.

3.      Fill Marketing Campaign Form and press Save button. Campaign will be saved if data is properly entered. Cancel button will abroad current Create New Marketing Campaign action. Reset button will empty input fields and all the data entered in fields will be lost.

Description of all input fields on this page as under:

Campaign name

Name of the campaign should be entered.

Campaign type

Campaign type should be selected from the drop down list.

Currency type

Currency type should be selected from the drop down list.

Start Date

Campaign start date should be selected from the calendar.

End Date

Campaign end date should be selected from the calendar.

Budget

Budget of the campaign should be entered.

Expected Cost

Expected cost of the campaign should be entered (not more than Budget).

Expected Revenue

Expected Revenue of the campaign should be entered

 

Actual Cost

Actual Cost of the campaign should be entered.

Objective

A few words describing the objective of the campaign.

Description

Enter few words describing the campaign.

Campaign Image

An image of the campaign can be uploaded.

Save

Press save button to save it.

Cancel Button

It will return you back to the campaigns main page.

Reset Button

To reset all the fields to default.

 

 

Campaign Settings

Overview

 The settings of the campaign created earlier under the Create Campaign window can be altered here.

How to do campaign setting?

1.      Click on marketing Tab.

2.      Click on settings tab.

3.      Press  icon to open Create New Settings Form.

4.      Enter data in all the fields and press save button to save it.

5.      To launch manually click on launch now button.

Description of all input fields on this page as under:

Customer

Drag and drop the customer or click on the customer and press arrow key to select the customer for the campaign.

Potential Customer Email

Enter the potential customer email addresses separated by ;.

Date

Select the date from the calendar.

Message

Enter the message you want to send to customers.

Loyalty

Overview

The Loyalty window is provided to display a list of all the loyal customers. The loyalty rules are set by the company. Any customer meeting the loyalty rule criteria at the time of placing a new order is then offered the Loyalty service/status[M2] .

How to create Loyalty?

1.      Click on Loyalty tab.

2.      Click on loyalty sub Tab.

3.      Press  icon to open Create New Loyalty Form.

4.      Fill Loyalty form and press Save button. Loyalty will be saved if data is properly entered. Cancel button will abroad current Create New Loyalty Form action. Reset button will empty input fields and all the data entered in fields will be lost.

Another rule can be added by clicking on Add Another Rule.

Description of all input fields on this page as under

Title

Enter the title name of the Loyalty.

Details

Enter few words describing the loyalty.

Launch

Select from the drop down list either you want to launch the loyalty manual or automatic.

 

Launch Date and time

Select the launch date and time from the calendar in case of automatic.

Status

Select the status from the drop down list.

Discount

Enter the amount (discount) which loyal customer would be offered.
