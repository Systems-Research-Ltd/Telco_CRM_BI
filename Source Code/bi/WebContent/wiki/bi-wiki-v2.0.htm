How to Login User

Overview

The User Login window is provided to allow the user to login to the application using his/her password. On successful authentication, the user is redirected to the dashboard.

How to Login?

1.      Enter User Name “provided by the account manager”.

2.      Enter Password “provided by the account manager”.

3.      Press Login. You will see the dashboard.

Description of all input fields on this page as under:

Login Name/Email

Enter the valid Login name/email.

Password

Enter the right password.

Login

Press this button to login.

Forgot Password

Press this button to recover forgotten password.

Dashboard

Overview

On successful login, the user is directed to the Dashboard window. This displays all the various widgets (accessible by the logged in user) which are useful for managing the Reports. At this level, various reports can be selected to be displayed like Bar Chart, Line Graph etc., as shown in the figure below.

 

The Dashboard has four widgets on this page where user can add different types of report.

How to Add Report to Widget?

Report can be added to widget by clicking on plus ‘+’ button. After click on plus button a new window pops out which looks like as under:

 

After clicking ‘+’ button above screen will be shown with the Reports list. Select the required report to be sown and then select report Type from the given option i.e. Table, Bar Graph, line Graph and Area Graph.

After selecting desired report and type click Submit to show report in Widget and click Cancel button to cancel the process.

Reports

Overview

The Reports tab is responsible for handling and managing all tasks associated to reports generation and record.

When the Reports tab is selected, two panels are available to the user; the Data Palette and the Design Palette. These provide the related features. To create reports, the user has to select the Data Source under the Data Palette panel.

 

 

Data Palette

Overview

The Data Palette panel is responsible for taking care of all data base related activities (for reports creation). This includes activities like creating Data Source, New Database Connection, applying Filters and other SQL Operations.

How to create Data Source?

1.      Press  icon to open Create New Data Source window.

2.      Fill Data Source Form and provide Data Source Name and select Database Connection from drop down list or create new connection. Provide Table Alias if required.

 

3.      After you select Connection from list the All Tables list will be populated with the available tables in database. Move selected tables to right side. Now click Next to proceed to next Screen. Cancel button will abroad current ‘Create New Data Source’ action.

 

4.      After you click Next button the above screen will be shown. This is used to add join clauses to different selected tables. After adding join clause click the Save button to add Data source. Cancel button will abort the action while Back button will take back to the previous screen.

Description of all input fields on this Table Screen are as under:

Data Source Name

Name of the Data Source should be entered.

Select Connection

Connection should be selected from the drop down list.

Shift Operations

Use these buttons to move selected tables from right to left or vice versa.

Table Alias

Table Alias should be entered if table alias is to be added before table names.

All Tables

This list shows all the available tables which are in selected Database Connection

Selected Tables

This list shows all the selected tables which are selected from the All Tables list.

Description of all input fields on this Join Screen are as under:

Join Clause

This Area shows the join clause which have been added on the tables.

Join Type

Select Join type from the drop down list like ‘Inner’.

Table

Select tables from the drop down List to apply Joins.

Columns

Select columns from the drop down List to apply Joins.

Join

Select Join to apply on tables like ‘Like’.

Apply

Click on apply button to Apply join.

Delete

Click on Delete button to delete selected Join Clause.

How to create a New Data Base Connection?

1.      Open Data Source Setting Form from Data Palette.

2.      Press  icon to open Create New Connection.

3.      Fill the form, select Database Manager from the List   and press Test Connection button. If Test is Successful click Save to save the New Connection. Cancel button will abroad current ‘Create New Database Connection’ action.

Description of all input fields on this page as under:

Connection name

Name of the Connection should be entered.

Database Manager

JDBC Driver

JDBC Driver should be selected from the drop down list.

Database

Host

Host must be entered.

Port

Poet must be entered.

Username

Username of the connection should be entered .

Password

Password should be entered

Save Password

Save Password checkbox must be selected if password is to be saved.

Default Schema

Default Schema of the Database should be entered.

Connection URL

This will show the Connection URL of the connection..

Save

Press save button to save it.

Cancel Button

It will return you back to the Data Source Settings Page.

 

 

 

 

 

Design Palette

Overview

Under the Design Palette, the users can actually design their reports according to their specific needs. The purpose of this panel or area is to group all Report Designing related tasks for better organization and management.

 

 

How to make a Report?

1.      In order to make a report you should have Data source.

2.      Than you should have a Data base Connection.

3.      After Selection Data source and Connection, the Data Palette will show all the available Database Tables and their Columns.

4.      Now Select the desired Columns and drag and Drop them in the Data Table Column area or in Graphical area accordingly.

5.      Now it will automatically display the Data in Table or in Graph From depending on in which area data was drag and dropped. Enter Report Name and Description. Provide Report summary if necessary. Upload Logo and Click Save Report to save it.

Description of all input fields and Action on this page as under:

Report Title

Report Title must be entered.

Report Description

Report Description must be entered.

Report Summary

Report summary must be provided if required.

Logo

Browse for Logo to be shown in Report.

Save

Click Save button to save Report.

Delete

Click Delete Button to Delete Report.

Cancel

Click Cancel Button to abort the Report.

Logo Checkbox

Check the Logo Checkbox to display logo in Report.

Limit Rows

Select no. of values of rows to be shown of report from drop down list.

Serial No. Checkbox

Check the Serial No. Checkbox to display Serial No. in Report.

Graph

Select type of graph in which to show Report from Graph Drop down list.

 

 

How to Add Logo to Report?

1.      In order to Show Logo on Report Logo Checkbox must be checked.

2.      Click on Edit Logo Button. A new Window will open.

 

 

3.      Click on Select Logo and Browse for the Image which should be used as logo. Than Save it. Logo will be displayed on Report when it will be saved.

 

Description of all input fields and Action on this page as under:

Select Logo

Click this button to browse for the Logo Image.

Save

Click this button to Save for the Logo Image for the Report.

Cancel

Click this button to cancel the current action.

 

 

 

 

 

 

 

 

 

 

 

 

 

Apply Filters and Formulas

Overview

This is provided to leverage the database related operations (like Order By, Group By) as well as apply various filters. This feature makes it more user friendly.

How to Apply Data Filter?

1.      Click on Add filters in Data filters section under Data Palette.

2.      This will open new window which will look like this:

3.      Select Table from Dropdown and column from list to apply filter.

4.      Now select Filter like filter and click Add. Provide value for filter. Now the current filter text area will show the sql for filter. Click on save to apply filter . Reset will clear all the fields and Cancel will close the window without applying filter. You can add another filter by clicking Add Filter button.

 

Description of all input fields on this Table Screen are as under:

Add Filter

Click on Add filter Button to add new Filter for Report.

Table List

Select Table from Drop down List on which filter is to be applied.

Column List

Column List will display all the columns of the Selected Table. Select column on which you want to apply filter.

Filter

Select Filter from filter Dropdown .

Add

Click on Add button to apply filter on Report

Save

Click on Save to Save current filter.

Cancel

Click on Cancel button to leave filter window.

Reset

Reset will clear all Fields

Delete

Delete will Discard the current filter from Report.

2.      Select the Report to be viewed. Selected Report will be shown.
