Company

Overview

The Company window is provided to list down the names of all the companies/clients/customers currently using the software.  Details for a particular company/client/customer of the software can be viewed by selecting that company’s name from the navigation pane provided on the left hand side of the window.

What is a Company?

A Company is the basic entity for UM. All rest of data belongs to company and overall flow circulates among this entity. All Franchises belongs to some company. At the time of login UM requires Company to keep data separate on the basis of company.

UM keeps data separate on the basis of Company. Every user created in UM belongs to some company. In same way Groups, Privileges varies company to company.

Edit

Data can be edited by clicking on edit button.

Delete

Company can be deleted by pressing delete button. Only after pressing delete confirmation button Company can be deleted.


 

Create Company

Overview

The Create Comapny window is used to create a company. Eventually, users, groups and franchises can be created, removed or exported for this newly created company.

 What is a Company?

A Company is the basic entity for UM. All rest of data belongs to company and overall flow circulates among this entity. All Franchises belongs to some company. At the time of login UM requires Company to keep data separate on the basis of company.

UM keeps data separate on the basis of Company. Every user created in UM belongs to some company. In same way Groups , Privileges varies company to company.

Can you complete the life cycle without Company?

Without the Company you can’t complete life cycle of UM.

How to create a Company?
 

  1. Login from Admin account.
  2. Click on Admin  Tab on top Tab bar under Header section





     
  3. Press  icon to open Create New Company Form.

  1. Fill Company Form and press Save button. Company will be saved if data is properly entered. Cancel button will abroad current ‘Create New Company’ action. Reset button will empty input fields and all the data entered in fields will be lost.

 

Description of all of the input fields on this page is as under.

 Company Title

Company Title is the actual name of Company. Later in application it will be used as a label to recognize this company.

Status

Status indicates the currently company is available for flow or not. Possible values are ‘Active’ and ‘In Active’. Active indicates company is available for flow. In Active indicates company is not available for flow. 

Email

Company email is the medium to establish communication with company. All notifications, Alerts and accounts will be manages/communicated via this email id.

Contact Number

Contact Number should contain valid phone number of the company. It would be used to contact the Company when needed.

Country/State/City

Company’s country/State/City should be selected from the drop down menu. It describes the company’s exact location.

Zip Code

Zip Code should be entered in accordance with the country.

Company Address

Valid company address should be entered. It is a place where a person or organization can be found or communicated with.

Company Details

Enter few words describing the company. This would help in understanding the nature of the company.

Contact Person

A specific person who will be contacted by the vendor.

User Name

Username using which the company will Login.

Password

Password using which the company will Login.

Confirm Password

Password should match the above given password.

Save

Press save button to save it.

Cancel Button

It will return you back to the company’s main page.

Reset Button

To reset all the fields to default.

What is Parameter?
